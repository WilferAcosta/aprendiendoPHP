Formulario CRUD en PHP
Este es un formulario que te permite realizar operaciones CRUD en una base de datos simulada a través de una API RESTful.

Abre el formulario en tu navegador web accediendo a la URL correspondiente (por ejemplo, http://localhost/formulario-crud/).

Ingresa los datos en los campos del formulario.

Utiliza los botones para realizar las operaciones CRUD:

✔: Enviar datos (Crear).
✎: Actualizar datos.
x: Eliminar datos.
⌨: Buscar datos.
↑: Cargar datos en el formulario para editarlos.
